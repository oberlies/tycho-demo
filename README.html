EclipseCon 2014 Tycho Tutorial

Setup/Prerequisites

  1. You need a Kepler (4.3) Eclipse classic SDK.
    • Start it, use Help > Install New Software... to install
    • "m2e - Maven Integration for Eclipse" from http://download.eclipse.org/releases/kepler/
    • "Eclipse 4 Core Tools" from http://download.vogella.com/kepler/e4tools
    • "Tycho Project Configurators" from http://repo1.maven.org/maven2/.m2e/connectors/m2eclipse-tycho/0.7.0/N/0.7.0.201302171659/

    For your convenience, we provide pre-packaged zips/tarballs with this installation in the root folder of the USB stick.
  2. Copy and unpack the Eclipse SDK archive for your platform and start it.
    NOTE: For the 64-bit versions you will need a 64-bit JVM installed
  3. Copy folder tutorial/ to your local hard drive, e.g. to C:\temp\tycho\ or ~/tycho/

Exercises Overview: Building an Eclipse 4 RCP application in five step-by-step exercises

The hands-on exercises will guide you through setting up a typical Tycho build for an Eclipse 4 RCP application.
  • We start from scratch with an Eclipse 4 sample application plugin,
  • Add a simple unit test to make sure the plugin works, and
  • Create a feature for grouping the plugins we want to deliver.
  • Then, we create a p2 repository/update site containing our feature(s) and bundles, and
  • Finally, we create a ready-to-use distribution archive of our RCP application.

Table Of Contents

The exercises consist of five steps:
  1. Build a Eclipse 4 RCP plugin
  2. Build and execute a simple test
  3. Add a feature
  4. Add a p2 repository
  5. Build a product (solution)

Each step has a sub-folder with stub content as a starting point.
The links above give detailed step-by-step instructions for each step.
The folder for an exercise is at the same time starting point for the current exercise and solution of the previous one.
If you want to catch up to a certain step during the exercises, just follow the recovery instructions provided in the corresponding step.

If you successfully installed everything under "Setup" above, you can now jump to the first exercise.
